# A Pluggable Hybrid Intrusion Detection Suggestion and Recovery Framework for Web Applications Using XGBoost, DNN and Random Forest

## Project Overview
This project implements a hybrid intrusion detection system for web applications using XGBoost, Random Forest, and Deep Neural Network (DNN). The system detects malicious traffic and provides recovery suggestions for improving security.

## Features
- Data preprocessing
- XGBoost classifier
- Random Forest classifier
- Deep Neural Network (DNN)
- Hybrid ensemble prediction
- Intrusion detection
- Recovery suggestion module
- Performance visualization
- Gradio web deployment
## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- TensorFlow/Keras
- XGBoost
- Gradio
- Matplotlib

## Project Files
- app.py
- preprocess.py
- train_xgb.py
- train_rf.py
- train_dnn.py
- hybrid_evaluation.py
- suggestion_recovery.py

## Deployment
Run:

python app.py

The Gradio interface launches locally and generates a public share link.
